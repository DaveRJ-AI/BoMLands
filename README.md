# Book of Mormon Internal Geography Map

This project builds a **text-derived internal map** of the Book of Mormon based solely on relationships described in the text.

## Core Principles

- No assumption of real-world geography
- All spatial relationships come from the text
- Ambiguity is preserved, not eliminated
- User-controlled toggles affect interpretation (e.g., UP/DOWN)

## Data Layers

1. Locations
2. References (verses)
3. Spatial Terms
4. Spatial Claims (relationships)
5. Events (missions, wars, migrations)

## Structure

- `/data/pilots/` → experimental datasets (current work)
- `/data/core/` → merged canonical dataset
- `/data/vocab/` → controlled vocabularies
- `/data/review/` → unresolved or ambiguous items
