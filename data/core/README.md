# Core Dataset

This folder contains the merged, chronology-aware Book of Mormon internal geography dataset.

## Current Status
- Alma 22 geography frame merged
- Alma 17–26 missionary overlay merged
- Alma 43–62 military overlay merged
- Chronology-aware schema enabled

## Chronology Periods
- jaredite
- pre_christ
- destruction
- post_christ

## Notes
- Most current merged records are tagged `pre_christ`
- Jaredite overlap has begun where relevant (for example, Desolation/Jaredite region linkage)
- Destruction and post-Christ layers will become much more important once 3 Nephi datasets are added
